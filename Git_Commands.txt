+++++++++++++++++++++++++++++++++++++++

Step-by-Step Git Beginner Tutorial
Step 1: Create a folder
mkdir myproject
cd myproject

Step 2: Initialize Git
git init

Step 3: Create file
echo "Hello" > test.txt

Step 4: Stage it
git add test.txt

Step 5: Commit it
git commit -m "Initial commit"

Step 6: Add remote-------------First time only
git remote add origin https://github.com/user/repo.git

Step 7: Push it
git push -u origin main
